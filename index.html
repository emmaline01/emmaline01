Snake Robot Locomotion: Directional Compliance

×

My Role:
I ran experiments comparing robot locomotion with/without various types of directional compliance locomotion by collecting, graphing, and analyzing data on feedback from sensors on the robot. I noticed correlations between torque sensor readings and when the robot hit obstacles, and developed a mass-spring-damper controls system that regulates the snake robot's amplitude based on those torque readings.

Project Context:
This snake robot requires pushing off of obstacles in order to propel itself forward, but complying with obstacles when they get in the way. Directional compliance determines when the robot should be pliable and when it should push against obstacles.

Cardistry Dashboard

×


I created this dashboard as a personal project to support myself through the process of practicing cardistry. I had extremely minimal knowledge of front-end/back-end technologies prior to this, and also wanted to explore making my own web app. It recommends and displays YouTube tutorials of new moves for me to learn using the Google Data API, and when I'm learning moves I can add, edit, or remove them from a Microsoft Azure database. Finally, the dashboard suggests combinations of moves for performing using features of the moves in the database and a Markov chain.
